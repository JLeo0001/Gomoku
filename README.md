# Gomoku · 五子棋

Purpur 1.21+ 五子棋小游戏插件。创建场地后全自动搭建棋盘与等待区，提供精美箱子 GUI 和智能 AI 对手，开箱即用。

## 特性

- **🎮 精美箱子 GUI** — 主菜单、场地浏览、观战列表全部图形化操作，无需记忆任何指令
- **⚔️ PvP & PvE** — 玩家对战 + 人机对战，AI 采用迭代加深 Negamax + α-β 剪枝，威胁检测 + 走法排序
- **🏗️ 全自动搭建** — 创建场地自动生成独立世界、棋盘、等待区、玩家站位，无需手动建造
- **💀 骷髅头颅棋子** — 白棋 = 骷髅头颅，黑棋 = 凋零骷髅头颅
- **👁️ 观战系统** — 旁观者模式 + 飞行，观战聊天隔离，可限制最大人数，对局结束自动传回原位
- **📦 背包保护** — 进入游戏自动保存背包/经验/状态，离开自动恢复，游戏内禁止破坏/伤害/掉落
- **⏱️ 全自动流程** — 大厅倒计时 → 回合计时 → 自动判胜 → 超时/弃权判负 → 棋盘清理
- **🔌 断线保护** — 玩家退出/被踢/切换世界自动判定弃权并恢复
- **⚙️ 配置驱动** — 修改 `config.yml` 后 `/gka reload`，所有棋盘自动重建
- **🌐 双语言** — 内置中文 + 英文，所有提示可自定义
- **📦 零依赖** — 无需资源包，无需任何前置插件

---

## 快速开始

1. 将 `Gomoku-1.2.0.jar` 放入 `plugins/`，启动服务器
2. `/gka create arena1` — 自动生成世界 + 棋盘 + 等待区
3. `/gomoku` 打开主菜单，点击即可游玩

## 指令

### 🎮 玩家指令 `/gomoku`（别名 `/gk` `/go`）

| 指令 | 说明 |
|---|---|
| `/gomoku` | 打开 GUI 主菜单（**推荐**） |
| `/gomoku gui` | 打开 GUI 主菜单 |
| `/gomoku join pvp [场地]` | 加入 PvP 匹配（无场地则自动选择） |
| `/gomoku join pve [场地]` | 加入 PvE 人机对战 |
| `/gomoku leave` | 离开当前对局 / 退出观战 |
| `/gomoku spectate [场地]` | 观战活跃对局 |
| `/gomoku version` | 查看版本 |

### 🔧 管理员指令 `/gomokuadmin`（别名 `/gka`）

| 指令 | 说明 |
|---|---|
| `/gka create <id>` | 创建场地（世界 + 棋盘 + 等待区全自动） |
| `/gka delete <id>` | 删除场地并卸载世界 |
| `/gka status [id]` | 查看场地状态 |
| `/gka list` | 列出所有场地 |
| `/gka set <id> <点位>` | 手动微调点位（lobby / spawn1 / spawn2 / spectator） |
| `/gka maxspec <id> <数量>` | 设置最大观战人数 |
| `/gka end <id\|all>` | 强制结束对局 |
| `/gka reload` | 重载配置 + 重建所有棋盘 |

## GUI 菜单

```
┌──────────────────────────────────────────────┐
│              五子棋 — 主菜单                    │
│                                              │
│   [⚔ PvP]  [☰ 浏览]  [📖 状态]  [☠ PvE]  [👁 观战]  [✖ 离开]  │
│                                              │
│   灰色玻璃镶边边框                               │
└──────────────────────────────────────────────┘
```

- **PvP 快速匹配** — 自动寻找空闲场地，等待对手
- **人机对战** — 自动寻找空闲场地，与 AI 对战
- **浏览场地** — 查看所有场地：🟢 空闲 / 🔴 使用中 / ⚫ 维护
- **观战** — 浏览活跃对局，点击即可观战（旁观者模式 + 飞行）
- **离开** — 退出当前对局或观战

## 配置

`plugins/Gomoku/config.yml`：

```yaml
board:
  size: 19                     # 棋盘大小（9–25）
  y-level: 64                  # 棋盘 Y 坐标
  surface-block: SMOOTH_STONE  # 棋盘表面方块
  grid-block: DARK_OAK_PLANKS  # 棋盘边框方块

timing:
  lobby-countdown: 30          # 大厅倒计时（秒）
  turn-timeout: 60             # 回合计时（秒）
  game-max-duration: 1800      # 最大对局时间（秒，30 分钟）

spectator:
  max-per-arena: 10            # 单场地最大观战人数
```

## 场地结构

```
           [白棋站位]          [黑棋站位]
          台阶平台              台阶平台

    ╔═══════════════════╗
    ║  平滑石 棋盘 N×N  ║
    ╚═══════════════════╝
       深色橡木边框

           [等待区]
       石砖平台 + 信标
```

场地使用独立世界（`gomoku_arena_<id>`），创建时自动生成：
- 平滑石棋盘 + 深色橡木边框
- 两侧玩家站位（台阶平台）
- 北侧等待区（石砖 + 信标）
- 世界预设：永昼、无天气、无生物生成、死亡不掉落

## 语言

`plugins/Gomoku/Language/zh_CN.yml` 和 `en_US.yml`，支持热重载（`/gka reload`）。

格式示例：
```yaml
prefix: "&8[&6五子棋&8]&r"
win: "{prefix} &6{player} &a获胜！"
gui:
  main:
    title: "&8五子棋 — 主菜单"
```
